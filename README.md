# 🦆 Duck Hunt 🦆
## Description
The user is looking down-range at a set of ducks that **need** to be hit. The mouse is used to aim and shoot at the ducks. Accurate shots are rewarded with explosive results and points are tallied up. Will you be able to beat the high score with limited time?!

## Installation

### Install VS Code + Ubuntu (follow instructions in document)
   - [WSL2 and Ubuntu on Windows 2025](https://github.com/user-attachments/files/27423348/WSL2.and.Ubuntu.on.Windows.2025.docx)
   - [Setup Instructions For Mac M1-M2](https://github.com/user-attachments/files/27423365/Setup.Instructions.For.Mac.M1-M2.docx)

### Clone git repository and open in VS Code
1. in Ubuntu terminal enter:
```
git clone https://github.com/Pethomps/CS151FinalProject.git game
```
2. to open in VS Code:
```
cd game
code .
```
### Compile the game
In VS Code run:
```
make game
```
### Run the game
In the VS Code terminal, run
   ```
./game
   ```
