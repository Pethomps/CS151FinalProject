## Description
The user is looking down-range at a set of ducks that **need** to be hit. Accurate shots are rewarded with explosive results and points are tallied up. Will you be able to beat the high score with limited time?!
---

## 🎯 How to Play
- **Left Click** -- Shoot
- Aim carefully -- you only get **one bullet** at a time!
- Hit as many ducks as you can before the timer runs out!

## Installation

### (1) Install VS Code + Ubuntu 
Follow instructions in relevant document:
   - [WSL2 and Ubuntu on Windows 2025](https://github.com/user-attachments/files/27423348/WSL2.and.Ubuntu.on.Windows.2025.docx)
   - [Setup Instructions For Mac M1-M2](https://github.com/user-attachments/files/27423365/Setup.Instructions.For.Mac.M1-M2.docx)

### (2) Download **SFML**  (version 3+).
Either [Download SFML](https://www.sfml-dev.org/download/) or use the CLI:  

Ubuntu/WSL: 
```bash 
sudo apt install libsfml-dev
```
Mac CLI: 
```bash
brew install sfml
```

### (3) Clone git repository and open in VS Code
1. in Ubuntu terminal enter:
```bash
git clone https://github.com/Pethomps/CS151FinalProject.git game
```
2. to open in VS Code:
```bash
cd game
code .
```
### (4) Compile the game
In the root directory run:
```bash
make game
```
### (5) Run the game
Then run
   ```bash
make run
   ```
