# CS151FinalProject

Show all local branches: `git branch`  
Show current branch: `git status`

## Git Instructions
1. Create branch: `git checkout -b [new branch]`
2. Publish remote
3. Move into a branch: `git checkout [branch name]` 
4. Make changes to functions
5. Connect new branch to Github repository: `git push --set-upstream origin [new branch]`
6. Stage your changes: `git add [filename]` or `git add .`
7. Commit to your changes: `git commit -m "message"`
8. Send your changes to Github: `git push`
